# Maya Python Tools
## 実行環境
　・Autodesk Maya 2024   
　・Windows, Mac (一部ツールはWindowsのみ対応です)

## 使用言語
　・Python 3.10.8

## 使用モジュール
　・7作品全てMaya 2024にデフォルトでインストールされているモジュールを使用しているため、追加インストールは不要です

## 実行方法
　・C:/Users/(user name)/Documents/maya/2024/scriptsもしくは  
　　C:/Users/(user name)/Documents/maya/2024/ja_JP/scriptsに、.pyファイルを入れる  
　・Maya 2024を起動  
　・スクリプトエディタを開き、左上の「ファイル」＞「スクリプトを開く」を選択し該当する.pyファイルを選択  
　・実行  
　・スクリプトエディタのPythonタブ内に直接スクリプトをペーストし実行することでもご利用いただけます

　・常識の範囲内でご自由にご利用ください、また利用において生じたトラブルや損害等の責任は一切負いません

## 各ツール概要
### Stairs Generator
　・**作品概要**：直線階段・らせん階段を生成するツール  
　・**制作理由**：階段を1からモデリングするのは大変な手間と時間がかかることから  

<details>
<summary>・使用方法 (直線階段)</summary>

1. 「生成方法の選択」から選択
2. 「段の生成方法の選択」から選択
3. 各数値を指定
4. 「階段の種類の選択」から選択
5. 生成する階段のフェースを表面のみにする場合、「階段の表面以外のフェースを削除」にチェック
6. 「踏板同士の重なった頂点をマージ」で踏板同士の重なった頂点がマージされた状態で生成されます
7. 「生成」をクリック、直線階段が生成されます
</details>

<details>

<summary>・使用方法 (らせん階段)</summary>

1. 「生成方法の選択」から選択
2. 「段の生成方法の選択」から選択
3. 各数値を指定
4. 「階段の種類の選択」から選択
5. 支柱も含めて生成する場合、「支柱を作成」にチェック、「中央支柱用の穴の半径」にぴったり収まるように支柱が作成されます   
   また、踏板の支柱と接する部分のフェースは自動的に削除されます
6. 生成する階段のフェースを表面のみにする場合、「階段の表面以外のフェースを削除」にチェック
7. 「踏板同士の重なった頂点をマージ」で踏板同士の重なった頂点がマージされた状態で生成されます
8. 「生成」をクリック、直線階段が生成されます
</details>

　・**制作理由**：新規フォルダを事前に用意する手間を省き、かつ手軽に書き出しを行いたかったことから  
　・**注意事項**：本ツールはWindowsのみ対応です  

<details>
<summary>・使用方法</summary>

1. まず始めに書き出したいオブジェクトを選択
2. ファイル名を入力、もしくは「そのまま」に指定
3. 書き出し先(ファイルパス)を指定
4. 新規フォルダを作成する場合「新規フォルダを作成」にチェックし、新規フォルダ名を入力
5. 書き出し」をクリック、書き出しが実行されエクスプローラーが自動で起動します
6. 「ダイアログボックスを開く」でダイアログボックスが開きます
7. 「エクスプローラーを開く」でエクスプローラーが開きます
8. 「ログのクリア」でログをクリアできます
</details>

### Object Visibility Options Tool
　・**作品概要**：オブジェクトを任意のビューでは非表示で、別の任意のビューでは表示できるように設定するツール  
　・**制作理由**：Mayaのイメージプレーンにあるオプション「カメラ越しの視点」をオブジェクトに対しても使用したいとのことから  

<details>
<summary>・使用方法</summary>

1. 非表示にしたいオブジェクトを1つ選択
2. 選択した状態で「選択中のオブジェクトを指定」をクリック
3. 非表示にしたいビューのチェックボックスにチェック、複数選択可
4. 「非表示を適用」をクリック、オブジェクトが選択したビューで非表示になります
5. 「非表示を全て解除」で全てのビューでオブジェクトが表示されるようになります
6. 非表示を適用したままツールを終了した場合、自動的に非表示が解除されます
</details>

### What Is The Working Units Tool
　・**作品概要**：現在の作業単位の確認や変更を行えるツール  
　・**制作理由**：チーム制作の際に、作業単位の確認を見落としサイズの異なるモデルを提出される事が度々あり、  
　　　　　　　きちんと確認を行なってほしいと感じたことや、より簡単に作業単位の変更が行えるようにと思ったことから  

<details>
<summary>・使用方法</summary>

1. 作業単位を変更する場合、「長さ」「角度」「時間」を指定
2. 「作業単位を変更」をクリック、作業単位が変更されます
3. 「作業単位の確認」で現在の作業単位が表示されます
4. 「プリファレンスを開く」でプリファレンスが開きます
</details>

### Set Random Transform Tool
　・**作品概要**：オブジェクトを指定した範囲でランダムに配置、回転、スケールを行うツール  
　・**制作理由**：オブジェクトを乱数的に人の手で配置するのは大変な手間がかかる上、どうしても偏りが出てしまうことから  

<details>
<summary>・使用方法</summary>

1. ランダムに配置したいオブジェクトを選択、複数選択可
2. 「配置方法の選択」から選択、  
   「完全にランダム」の場合World座標基準になり、「現在の座標を基準に」の場合Relative座標基準になります
3. 「座標の範囲」「回転の範囲」「スケールの範囲」それぞれ最小値と最大値を指定、この指定した範囲でランダムになります
4. 座標や回転などの特定の値をゼロにする場合、「値をゼロに設定」のチェックボックスにチェック
5. 「ランダム配置」をクリック、ランダムに配置されます
6. オブジェクトを複製する場合、複製するオブジェクトを選択した状態で「複製回数」を指定し、「複製」をクリック
</details>

### Cube To Sphere Generator
　・**作品概要**：立方体をベースにポリゴンが均等に配分される球を生成するツール  
　・**制作理由**：Mayaのデフォルトで作成できるSphereではポリゴンの配分が上下に偏っており、  
　　　　　　　そうでない均等に配分された球が必要な際に一手間かかるとのことから  

<details>
<summary>・使用方法</summary>
  
1. 「生成方法の選択」から選択
2. 各数値を指定、  
   「球のセグメント数/分割数」は、  
   ベベルを選択した場合セグメント数として扱われ、スムースを選択した場合分割数として扱われます
4. 「球全体をハードエッジ」でハードエッジの球になります
5. 「ヒストリを残さず生成」でヒストリ情報が残らず生成されます
6. 「生成」をクリック、球が生成されます
</details>
